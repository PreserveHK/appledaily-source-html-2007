漏網新機　W660i背脊紋身出晒位

上星期iPhone贏晒，連大牌子都要行埋一邊回避！為尋訪iPhone蹤迹，用家甚至冇乜留意上個禮拜究竟有咩機登場。不妨話你知，其實上星期忙到嘔，除iPhone之外，索愛、摩記亦有新機種誕生，其中SEW660i更採用機背紋身搞出位。點都好，iPhone氣勢咁勁，回避一下都係好正常！

記者：陳國輝

W660i酒紅色最Grand

SE好耐冇推出過直身版WalkmanPhone，W660i採用簡約設計以及特別色彩做賣點，酒紅色機身配搭一條金邊固然搶盡風頭，但背脊嘅凹凸雕花紋身更叫人難忘。整體分唇唔算好勁，只有200萬像素相機、Walkman專用鍵以及M2記憶卡槽，算是中規中矩。

PROFILE
3G及GSM三頻、200萬像素相機、A2DP、FM收音機、支援M2記憶卡
SonyEricsson（82038863）

Z6抄歌2秒

玩音樂唔一定係SE嘅天下，摩記上星期亦推出一款強玩Music嘅Z6。接駁上太陽眼鏡般嘅S9藍芽立體聲耳機固然好型，但講到方便，始終唔及佢跟機送嗰張《MediaPlayer11Upgrade》軟件咁正！一經駁落電腦，僅2秒即傳送一首歌到手機，快到離譜。

PROFILE
GSM四頻、Linux作業系統、26萬色芒、200萬像素相機、A2DP、支援microSD
Motorola（25063888）

i300添加GPS

台灣近排動靜多多，除HTC推出Touch捽芒功能力撼iPhone，另一品牌GSmart亦加入陣線，旗下i300以內置GPS全球定位系統最搶鏡，重量僅116g，同類型當中屬輕量級別。佢仲採用最新嘅WindowsMobile6作業系統，成副架生夠新淨。香港僅推出黑色，台灣則推出白色。

PROFILE
GSM三頻、WindowsMobile6系統、MarvellPXA270520MHz、內置GPS、WLAN
誠達電訊（23312348）

S600i一味夠平

大陸所謂嘅PDAPhone氾濫，搞到人人以為唔使$1,000就玩到智能機，簡直天方夜譚！台灣另一品牌iDo就趁勢推出真正叫「平」嘅PDA手機，呢款S600i雖則功能麻麻，例如採用舊版本作業系統，外形普通，但勝在售價$2,290，咁嘅價錢其他牌子好難搵。

PROFILE
GSM三頻、WindowsMobile5.0系統、FreescaleiMX21CPU266MHz、miniSD
Vision（31642288）
