陳敏兒𠱁仔食豆腐　一味靠剁

阿仔成日揀飲擇食，每次叫佢食菜食魚就嗌救命，冇晒佢符！今日遇上陳敏兒教路：「煮餸要用掩眼法，將食物剁碎再撈埋其他材料，唔鍾意嘅食物都睇唔到！」此外，仲有幾位廚藝高囗嘅媽咪示範得獎便當，1份肉、2份菜、3份五穀，營養均衡得來賣相仲極之可愛，阿仔睇到都流晒口水！

記者：趙珮倩
攝影：林栢鈞
場地：港燈家政中心

日前港燈家政中心舉辦了「愛心便當」創作比賽，評判之一的陳敏兒就帶埋囝囝一齊下廚，不少小朋友都會偏食，她的兒子也不例外，不過就有妙法對付﹕「佢最唔鍾意食豆腐，咁我就將豆腐剁碎混入肉類：又例如佢唔鍾意齋食薯仔，但加入蝦米撈勻，再煎香成薯餅，咁又會食唔停口！煮餸最緊要都係花啲心思，啲細路就肯食﹗」

健康便當最緊要有1份肉、2份菜、3份五穀。

一個好味便當需要有小朋友鍾意食到的味道。

五餅二魚

0hr20min
2人份量

材料

蓮藕/薯仔/南瓜各5両、免治豬肉2両、蝦米1湯匙、沙鯭仔2條、生菜5大片、葱1條

調味料﹕鹽/糖各1/4茶匙、生抽1湯匙、生粉2湯、麻油/胡椒粉少許

做法

1.蓮藕/薯仔/南瓜刨絲；蝦米切幼粒。

2.免治豬肉/蓮藕/薯仔/南瓜/蝦米加入調味料和葱拌勻至起膠，將肉糰分為5份壓成餅狀。

3.燒熱油，將肉餅和沙鯭仔以慢火煎至熟透，放在生菜上即成。

貼士

豬肉用刀切碎粒比用攪拌機攪碎更有口感。

快樂愛心親子便當

0hr30min
1人份量

材料

白飯3/4碗、鱈魚粉1包、雞蛋1隻、番茄1個、三文魚50克（切粒）、鮮果/肉鬆/紫菜各適量、鳴門卷4片、青瓜1條、紅米/葵花籽/亞麻籽/青豆/粟米/鹽各少許

做法

1.先將熱飯加入鱈魚粉拌勻。

2.將一半鱈魚飯倒入倒模中，加上適量肉鬆做夾心，再鋪上鱈魚飯，從倒模中取出飯糰，加上紫菜/青豆/粟米點綴。

3. 將番茄的頂部切開，挖空備用。

4. 雞蛋加鹽拌勻，燒熱鑊，下蛋液/紅米/葵花籽快兜，最後下三文魚略炒，放入番茄中，撒上亞麻籽點綴。

5. 鳴門卷放入沸水中汆水過冷河，瀝亁；青瓜切成條狀，兩者與鮮果一同伴碟即可。

貼士

飯糰以鱈魚粉調味，不但能豐富白飯味道，其淡淡的粉紅色亦令賣相加分。

太陽出來了

0hr30min
1人份量

材料

南瓜1個、菠菜麵1個、牛肉/金菇/鮮蘆笋各適量、雞蛋1隻、生粉少許

調味料：橄欖油1湯匙、甜豉油1湯匙、日式芝麻醬2茶匙、鹽1/4茶匙

做法

1.南瓜去皮去籽，切成太陽花形，隔水蒸10分鐘。金菇/鮮蘆笋加入少許鹽灼熟備用；

2. 菠菜麵煮熟，加入適量的橄欖油/甜豉油拌勻，放入太陽花內。

3. 牛肉薄片鋪平，捲入金菇，以生粉水封口煎熟。

4. 雞蛋煎熟切絲，放上菠菜麵砌成花蕊，以蘆笋/金菇牛肉襯托。吃時伴以日式芝麻醬。

貼士

菠菜麵煮熟後要盡快加入橄欖油拌勻，否則很容易會黏在一起。

青蛙王子

0hr30min
1人份量

材料

雞蛋2隻、魚肉3安士、番茄仔2個、生菜1片、甘笋/紫菜各少許、蘆笋1條、紅米/白米各半杯、粟粉/鹽各少許

做法

1.白米/紅米洗淨、煮熟備用。

2. 雞蛋加入鹽/粟粉拂勻，煎成薄皮。

3.將魚肉薄塗在蛋皮上，輕輕捲圓卷，用中火蒸10分鐘，切件。

4.甘笋/蘆笋汆水，與紫菜/番茄仔/生菜點綴即可。

貼士

紅米質地較硬，煮食前需浸透最少兩小時，煲出來的紅米飯才夠軟身。
