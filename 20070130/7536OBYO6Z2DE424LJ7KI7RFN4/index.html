REVIEWS相機：
動態範圍校400%　剔走背光黑雜訊

嘩！黑白對比咁強，好難影，唔係黑，就係爆晒光。對於唔識影相的人，今次有救！富士單鏡機S5Pro標榜低雜訊和兼顧黑位和白位Detail（動態範圍），即使背光都可唔用反光板閃燈，人和背景一樣咁清晰，環境越差，影出嚟嘅相越有Fashionfeel。

記者：馬靜兒　攝影：陳偉強
Model：Isis

黑白平衡夠細緻

身邊好多同事都好期待，而眾人等到頸都長，今日終於等到它在香港推出，當然想即時試機，S5Pro硬件跟NikonD200幾乎一樣，加上富士本身的軟件和CCD，所以部份功能也跟D200很相似。雖然說很相似，但當然有分別，S5Pro最大的賣點是它的感光元件－SuperCCDSRPro，加埋RealPhotoProcessorPro，兩者可以減低用高ISO時產生的雜訊，又可以平衡高低光位的差異，無論光位或暗位的細緻都能保留。這是以往很多攝影師遇到的問題，而其他相機無法解決的，S5Pro表現較其他單鏡反光相機都出色，連用高ISO出現的雜訊都少咗同幼細咗，而且S5Pro貫徹富士相機色彩鮮艷的優點，令拍出來的相片顏色很鮮艷。

單一光源

快門：1/250
光圈：8
ISO800

試雜訊明顯較幼細

要試雜訊，黑位最明顯，Model穿黑色裙，站在黑色布簾前，光源只靠一支燈。
今次用高ISO，雜訊一般都會較多，黑位出現一粒粒的白點。今次在室內Studio用ISO800影相，特意只在右方打燈玩黑白位，雖然黑位依然有雜訊，但跟其他相機相比，雜訊明顯較少、較幼細。

動感快拍

快門：1/250
光圈：11
連拍模式

試連拍超慢……

一般單鏡機都有連拍的功能，今次叫Model跳起，以連拍方式拍下躍起的照片。
同級的NikonD200則可連拍無限張，Preview唔使一秒即有相睇。不過，S5Pro每次只可連拍7張，Preview要等足一秒才看到相，這一項差啲喎！

五顏六色

快門及光圈：
Program

試顏色鮮色夠豐富

富士出名顏色夠Rich，今次只用Program拍下Model粉紅色的外套，顏色甚至較外套本身的顏色更鮮更艷；Model的膚色卻自然柔和，背景的樹葉和藍天色調十分豐富。

大背光

快門及光圈：
Program

試「動態範圍」
高低光位平衡好

背光影相最常見的問題是黑位Detail清楚，白位就曝光過度；而白位清楚，黑位就會無晒Detail，兩者要取得平衡並不容易。平日背光影相，一定要加閃燈及反光板，否則主體會黑麻麻，背景就爆晒光。今次開啓「動態範圍」，調校至400%，唔用反光板、唔用閃燈，結果Model面容清晰，連短裙的Detail位都影得清清楚楚。

特有對焦
及人面檢視

影完人像，出了Preview，按機背左下角就可以即時檢視人物面部表情，Check吓有無眨眼，對焦有無對錯。不用先放大照片，再慢慢推至面部位置，再按一下，就會Check埋相中其他人的面部表情，最多可Check10塊面，有點似卡片機的Facedetection，但S5Pro要影完才可快速Check是否夠Sharp。

蘋果攝影師試用

Wilson（左）：顏色豐富，喜歡高低光位可以調校。
Kenny（右）：連拍太慢，Preview又慢，影相唔夠爽，但適合影婚紗照，可以同時兼顧白色婚紗和背景的黑位，因為富士機影的相顏色較鮮，影花都唔錯。

阿強：這種機唔可以操勞，因為連拍唔夠快，但雜訊控制得比其他單鏡數碼相機好。
